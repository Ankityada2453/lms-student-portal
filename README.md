# Student Portal (LMS) – React Assignment

This is a frontend-only **Student Portal UI** built using **React.js** for a MERN Stack Developer assignment.

##  Features

* Student Login Page
* Dashboard with Overview Cards
* My Courses Page
* Assignments Page
* Profile Page
* Responsive Design
* Static / Mock Data (No Backend)

##  Tech Stack

* React.js (Functional Components)
* React Router DOM
* CSS

##  Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/lms-student-portal.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd lms-student-portal
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4 Run the App Locally

```bash
npm run dev
```

The app will run at `http://localhost:5173`

##  Deployment

The app can be deployed using **Vercel** or **Netlify**:

1. Push the project to GitHub
2. Import repository in Vercel/Netlify
3. Build Command: `npm run build`
4. Output Directory: `dist`

## Notes

* This project uses **mock/static data only**
* No backend or authentication logic is implemented
* Designed for technical evaluation purposes
